Request-Response Lifecycle

The request-response lifecycle is the sequence of steps which occur when your browser makes a request to a server; the server processes that request and queries a database; the server sends back a response; and, the client processes and renders the response.

DNS Resolution

DNS Resolution is the process of locating an IP address for a given domain name; this process ensures that a device can connect to the correct server hosting the site.

TCP Handshake

TCP Handshake is a 3-step process used to establish a connection between the client and a server over the Transmission Control Protocol.

TLS Handshake

TLS Handshake is a 3-step process used to establish a secure & encrypted connection between a client and a server over the Transport Layer Security Protocol.

Request Construction

Request Construction is the process of a building an HTTP request that the client sends to a server, assembling a request method, URL, header, and optional body containing data.

Request Sending

Request Sending refers to the duration it takes for an HTTP request to travel from the client to the server after the request has been constructed.

Request Parsing

Request parsing refers to the process of breaking down a request into its components, such as headers, parameters, and body content to handle the client's intent.

Middleware Execution

Middleware execution involves processing requests and performing specific tasks (such as parsing, authentication, and validation) before the request reaches the main application logic.

Business Logic Execution

Business logic execution refers to the core functionality that defines how an application processes data and handles user interactions, applying specific algorithms and calculations to accomplish a goal.

Database Execution

Database execution refers to the process of retrieving data from a database, including DNS resolution, TCP connection, authentication, query execution, and data retrieval.

Response Construction

Response construction refers to the process of formatting data into a client-expected format (e.g. JSON), setting appropriate status codes and headers.

Response Sending

Response Sending refers to the duration it takes for an HTTP response to travel from the server to the client after the response has been constructed.

Response Parsing

Response parsing involves the client interpreting and converting the raw data from the response into a usable format.

Scripting

Scripting refers to the execution of JavaScript code on a web page, used to manipulate the Document Object Model, handle user interactions, and perform other various tasks.

Reflow

Reflow is the process by which the browser recalculates the layout to ensure that everything is correctly positioned and sized.

Repaint

Repaint is when the browser updates the visual appearance of a page after the layout has been calculated, updating styles, colors, and content on the screen.
